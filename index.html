Grid Challenge

Test your visual memory and pattern recognition skills

How to Play

Memory Phase

Watch highlighted dots (in black) and memorize their positions

Symmetry Phase

Determine if two grids are symmetrical (YES/NO)

Recall Phase

Click dots in the same sequence you saw them highlighted

Scoring System

Earn 1 point only when ALL phases in a round are completed correctly. No partial credit awarded.

Game Duration: 6 Minutes
